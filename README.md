# Requirements

**Gradle 8.+**

**JDK 17**

**Kotlin v1.9.+**

**Java 1.8**

**Target API 34**

**Supported API level 21+ (Android 5.0 Lollipop).**

# Features

- Audio call
- Audio call upgradable to video call
- Video call
- Chat
- Collaborative whiteboard
- File sharing in call
- Screen sharing in call:
    - App only
    - System wide
- Call Recording
- Virtual Background
- External Camera
