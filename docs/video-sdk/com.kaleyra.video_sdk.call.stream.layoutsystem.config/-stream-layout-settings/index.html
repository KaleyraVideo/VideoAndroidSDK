val defaultCameraIsBack: Boolean = false

Indicates whether the default camera should be the back-facing camera. This setting can be used to determine the initial camera selection of the local user. Defaults to false, meaning the front-facing camera is the default.

val isGroupCall: Boolean = false

Indicates whether the stream layout is being used in a group call scenario. This setting might affect how streams are prioritized or displayed, as group calls often have different layout requirements than one-on-one calls. Defaults to false.
